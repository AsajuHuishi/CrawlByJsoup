# 简介
这是一个基于Jsoup的用来爬取网页上图片并下载到本地的Java项目。


# 环境
<ul>
<li>JDK 1.8</li>
<li>IntelliJ Idea 2020</li>
<li>Jsoup 1.13.1</li>
</ul>



# 目录结构
```bash
├─saveImage
│  ├─喵咕君QAQ(KH3)
│  └─小逝lullaby
└─src
    └─indi
        └─huishi
            ├─service   // Crawl.java
            └─utils		// CrawlUtils.java
```
